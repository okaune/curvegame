# CurveGame

Try [CurveGame Aplha 2.0](http://olekristian.com/projects/curvegame/)

Based on and inspired by [Curve Fever](http://curvefever.io/).

Controls for local multiplayer
* Player 1: ←, →
* Player 2: A, D
* Player 3: B, M
